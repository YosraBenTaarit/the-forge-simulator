## 🎯 Vision
The Forge is a professional-grade simulation environment designed to bridge the gap between academic theory and industry reality. We replace static tutorials with "Live-Fire" missions where juniors must navigate legacy codebases, resolve production outages, and handle business pressure.

### 🚩 The Problem
CS graduates know syntax but fail in real-world "Chaos." Companies spend 6 months and $50k onboarding juniors who aren't ready for legacy code, messy logs, and production outages.

## 🏗️ Technical Architecture (v0.1)
The Forge is built on a "Digital Twin" model:
- **Runtime:** Browser-based Node.js environment via WebContainers.
- **Chaos Engine:** A custom event-bus that triggers random system failures (Network latency, Memory leaks).
- **AI Mentorship:** Gemini 1.5 Pro integrated via Vercel AI SDK to provide "Senior-level" code reviews.
- **Observability:** Custom React-based monitoring dashboard (Mock Grafana).
